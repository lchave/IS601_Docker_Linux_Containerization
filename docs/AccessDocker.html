How to log into your Docker container?



Before following these instructions please ensure you have already installed the following software:

  • Docker

After the software is installed, you need to ensure that the Docker application is running properly on your computer.

  1. Launch the command prompt.

    2. You can test that Docker is running properly by executing the following command:

    docker run hello world

    The output of these command should be as follow

  2. Deploy your Docker container
    • Download the image file you would like to use (i.e.)
      • docker pull Ubuntu
  3. Login to the Docker container by executing the following command:
    4. docker run -it ubuntu:latest /bin/bash

    Let's break down the syntax of this command to understand how does this enable you to access the container

    • run - This will ensure to run the container if it is not running yet. If the container is already running you could replace run with exec
    • -it - this parameter specifies that you would like to launch and interactive session
    • ubuntu:latest - this is the image name that you would like to run on the container. If the container is already running you could replace this with the container name. In oder to get a list of the existing containers run the docker ps command.
    • /bin/bash - this represents the command you would like to run within the container, in this scenario this command will open a shell session
  4. Once you have access to the interactive session to your container you could use any of the learned Basic Linux Commands in order to navigate through your container.
